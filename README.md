《芒果小镇》博客
===========

本博客遵循MIT开源协议。

##如何组建出我的博客
- Jekyll: 静态网站模版引擎
- Github Pages: 挂载博客的服务器
- qiniu: 将静态图片都存放于又拍云上
- grunt: 用于网站的静态文件自动合并压缩，并部署
- 加速乐: DNS解析与线路优化
- React: 使用React来组织“[所有文章](http://www.emsuits.com/articles.html)”页面
